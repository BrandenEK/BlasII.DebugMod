# BlasII.DebugMod

---

## Hitbox Viewer
Press F2 to toggle hitbox viewer
- Grey - Inactive
- Green - Geometry
- Cyan - Player
- Red - Enemy
- Magenta - Damageable
- Blue - Trigger
- Yellow - Other

---

## Free Cam
Press F4 to toggle free cam mode
- Arrow keys can be used to move the camera around
- Hold left control to increase the speed
